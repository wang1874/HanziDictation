# 汉字听写 App 本地开发环境配置指南

## 一、所需软件

| 软件 | 版本 | 下载地址 |
|------|------|----------|
| Node.js | 18.x 或 20.x LTS | https://nodejs.org/ |
| Java JDK | 17 (JDK 17) | https://adoptium.net/ |
| Android Studio | 最新版 | https://developer.android.com/studio |
| VS Code | 最新版 | https://code.visualstudio.com/ |

---

## 二、安装步骤

### 1. 安装 Node.js

**Windows:**
- 下载 `.msi` 安装包
- 双击运行，一路 Next 即可
- 安装完成后，打开 CMD 输入 `node -v` 验证

**Mac:**
```bash
# 使用 Homebrew 安装
brew install node@20

# 验证安装
node -v
npm -v
```

---

### 2. 安装 Java JDK 17

**Windows/Mac:**
1. 访问 https://adoptium.net/
2. 下载 **JDK 17** (Temurin 17)
3. 安装完成后配置环境变量：

**Windows 环境变量配置:**
```
JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-17.0.x.x
PATH 添加 = %JAVA_HOME%\bin
```

**Mac/Linux 环境变量配置:**
```bash
# 编辑 ~/.zshrc 或 ~/.bashrc
export JAVA_HOME=/Library/Java/JavaVirtualMachines/temurin-17.jdk/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH

# 生效配置
source ~/.zshrc

# 验证
java -version
```

---

### 3. 安装 Android Studio

#### 3.1 下载与安装

1. 访问 https://developer.android.com/studio
2. 下载 Windows/Mac 安装包
3. 安装时勾选：
   - ✅ Android SDK
   - ✅ Android SDK Platform-tools
   - ✅ Android SDK Build-tools
   - ✅ Android Emulator (可选)

#### 3.2 配置 Android SDK 环境变量

**Windows:**
```
ANDROID_HOME = C:\Users\你的用户名\AppData\Local\Android\Sdk
PATH 添加 = %ANDROID_HOME%\platform-tools
PATH 添加 = %ANDROID_HOME%\tools
```

**Mac:**
```bash
# 编辑 ~/.zshrc
export ANDROID_HOME=~/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/platform-tools
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin

source ~/.zshrc
```

#### 3.3 验证 Android SDK

```bash
# 验证 SDK Manager
sdkmanager --version

# 如果提示找不到 sdkmanager，手动找到路径添加到 PATH
# Windows: C:\Users\用户名\AppData\Local\Android\Sdk\cmdline-tools\latest\bin\
```

## 三、配置项目

### 1. 克隆项目代码

```bash
# 如果还没有克隆过
git clone https://github.com/wang1874/HanziDictation.git

# 进入项目目录
cd HanziDictation
```

### 2. 安装项目依赖

```bash
# 确保网络正常（可能需要 VPN）
npm install
```

### 3. 生成 Android 原生项目

```bash
npx expo prebuild --platform android
```

---

## 四、启动开发服务器

### 开发模式（热重载，推荐调试用）

```bash
# 启动 Metro bundler（JavaScript 热重载服务）
npx expo start

# 或者
npm start
```

启动后会出现 QR 码，扫码可以在手机上预览。

### 预览效果

**方法一：在 Android 手机/平板上预览**
1. 手机安装 Expo Go App
2. 手机和电脑连接同一 WiFi
3. 用 Expo Go 扫码

**方法二：在模拟器上预览**
1. 打开 Android Studio
2. 启动一个 Android 模拟器
3. 在终端运行：
```bash
npx expo start
# 按 'a' 键启动 Android 模拟器
```

## 五、调试

### 使用 VS Code 调试

1. 安装 VS Code 的 React Native 插件：
   - **React Native Tools** (Microsoft)

2. 配置调试：
   - 按 F5 选择 "Debug Android"
   - 或创建 `.vscode/launch.json`:

```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Debug Android",
      "request": "launch",
      "type": "reactnative",
      "platform": "android"
    }
  ]
}
```

3. 设置断点，点击 F5 开始调试

### 常用调试快捷键

| 功能 | 操作 |
|------|------|
| 重新加载 | 按 `r` 键 |
| 打开开发者菜单 | 按 `d` 键 |
| 退出 | 按 `q` 键 |

## 六、打包 APK

### 调试版 APK（快速测试用）

```bash
cd android
./gradlew assembleDebug
```

APK 位置: `android/app/build/outputs/apk/debug/app-debug.apk`

### 发布版 APK（正式发布用）

```bash
cd android
./gradlew assembleRelease
```

APK 位置: `android/app/build/outputs/apk/release/app-release.apk`

> 首次构建会下载 Gradle 和依赖，可能需要 10-20 分钟。

## 七、常见问题

### Q1: npm install 报错

```bash
# 清除缓存重试
npm cache clean --force
rm -rf node_modules
npm install
```

### Q2: Gradle 构建失败

```bash
# 清除 Gradle 缓存
cd android
./gradlew clean
./gradlew assembleDebug
```

### Q3: JAVA_HOME 设置无效

- 确保 JDK 17 已正确安装
- 重启终端或电脑
- 检查环境变量拼写是否正确

### Q4: Android SDK 找不到

- 打开 Android Studio
- File → Settings → System Settings → Android SDK
- 查看 SDK Path 并确保环境变量正确
